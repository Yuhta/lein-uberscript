# lein-uberscript

A Leiningen plugin to create standalone script embedded with uberjar.

## Usage

Add [![Clojars Project](http://clojars.org/lein-uberscript/latest-version.svg)](http://clojars.org/lein-uberscript) to `:plugins`.

    $ lein uberscript
